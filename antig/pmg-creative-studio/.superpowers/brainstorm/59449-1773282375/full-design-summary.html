Re-open editor flow
  1. maskDataUrl exists? → load as initial mask state
  2. No maskDataUrl? → derive from extractedImageUrl alpha channel
  3. User refines → new maskDataUrl + new extractedImageUrl
Sync safety
  • No zoom/pan in v1 — eliminates coordinate transform bugs
  • Both canvases locked to naturalWidth × naturalHeight
  • path:created event-driven mirroring (not polling)
  • Dimension mismatch assertion → blocks Apply if diverged
